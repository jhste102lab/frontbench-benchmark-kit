# Security

This repository contains public benchmark materials only. It should not contain production credentials, private API endpoints, user data, model provider tokens, deployment secrets, or private FrontBench application code.

If you find sensitive information in this repository, open a private security report on GitHub if available. If private reporting is unavailable, open an issue with a minimal description and do not include the secret value.

The HTML validator is designed for local inspection of benchmark artifacts. Treat model-generated HTML as untrusted content. Do not open generated pages in a browser profile that contains sensitive sessions unless you have reviewed the file first.
